## 🌐 3. 라이브 URL 만들기 (10분, 한 번만)

팀원들이 코드 없이 브라우저에서 바로 볼 수 있게:

1. https://vercel.com 접속 → **Continue with GitHub** 로 로그인
2. **Add New... → Project**
3. `5jseung/eco-prompt` 저장소 → **Import**
4. Framework Preset: `Vite` (자동 감지됨, 그대로 두면 됨)
5. (선택) **Environment Variables** 펼치고 `GEMINI_API_KEY` 추가 → 본인 키 입력
   - 안 넣어도 앱 동작은 함, 단지 Gemini 정제 단계가 빠지고 로컬 프롬프트 초안만 나옴
   - 발급: https://aistudio.google.com/apikey
6. **Deploy** 클릭

1~2분 기다리면 `eco-prompt.vercel.app` 또는 `eco-prompt-xxx.vercel.app` 같은 URL 받음. 이게 팀원들한테 공유할 링크.

⚠️ 이 뒤로 `main`에 push할 때마다 자동으로 재배포됨. 매번 손댈 필요 없음.

## 🧪 5. 직접 한 번 테스트해보기 (5분)

링크 받은 다음 본인이 먼저 클릭해서 다음 항목 체크:

- [ ] 페이지 로드되고 헤더에 "Interactive Exhibition Genie" 보임
- [ ] 오른쪽 아래에 슬픈 물범이 움직이며 떠 있음 (배경 투명)
- [ ] 칩 몇 개 클릭하면 → 진행률 바 차오름
- [ ] 한 섹션 완성하면 → 카드 초록 테두리 + "완료" 표시 + 물범이 축하 애니메이션
- [ ] 모든 섹션 채우면 → 물범 활짝 웃는 표정 유지
- [ ] '프롬프트 생성하기' 클릭 → 섹션 8개 전부 접힘 + 하단에 영문 프롬프트 등장
- [ ] 접힌 카드 클릭 → 다시 펼쳐서 수정 가능
- [ ] '복사' 클릭 → 메모장에 붙여넣어서 확인
- [ ] (Gemini 키 설정한 경우만) 출력된 영문이 100~150 단어짜리 자연스러운 문장

뭔가 이상하면 단톡방에 스크린샷이랑 같이 알려달라고 하면 됨. (오류 메시지가 화면에 빨갛게 뜨면 그것도 같이.)
